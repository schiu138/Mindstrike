# ⚡ MindStrike

A browser-based educational RPG where answering questions correctly blocks enemy attacks and strikes back. Build streaks, slay monsters, and conquer knowledge — from kindergarten to doctoral level.

**[▶️ Play Now](https://schiu138.github.io/mindstrike/)**

---

## 🎮 How to Play

1. **Choose your level** — Kindergarten through Master/Research
2. **Pick your subjects** — each level unlocks new areas
3. **Set your battle duration** — 1 to 20 minutes
4. **Strike!** — Answer correctly to block attacks and deal damage. Wrong answers = take damage. Build streaks for up to 5× bonus multipliers.

## ✨ Features

- ⚡ Turn-based defend/attack combat
- ⏱️ Circular countdown timer (green → yellow → red)
- 🔥 Streak multiplier up to 5×
- 🔊 Text-to-speech on every answer
- 📖 Learn More link after every question
- 👁️ Eye health reminder for long sessions
- 📱 PWA — installable on iPhone and Android
- 🌐 Works offline after first load

---

## 📱 Install on Your Phone

### Android (Chrome)
1. Open the link in Chrome
2. Tap the **Install** banner, or tap ⋮ → **Add to Home screen**

### iPhone (Safari)
1. Open the link in Safari
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen → Add**

---

## 🚀 Deploy to GitHub Pages

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to **main branch / root** → Save
4. Live at `https://YOUR-USERNAME.github.io/mindstrike/`
